# 其他项目概述 {#overview-of-other-projects}

“其他”分类为你提供关于各类非 Geyser 或 Floodgate 相关项目的信息。
这其中既包含社区项目，也涵盖 GeyserMC 团队开发的其他项目（例如 Hurricane 或 Hydraulic）。
