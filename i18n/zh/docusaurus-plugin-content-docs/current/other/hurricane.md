# Hurricane

Hurricane 是一个 Paper/Spigot 插件，以及 Fabric 模组（非官方移植版），用于修复一些不修改服务端就无法解决的漏洞。

## 修复内容 {#fixes}
每个修复方案的问题都列在 Hurricane 配置文件中。**请仔细阅读，因为本插件中的修复方案可能会被用于恶意目的。**

- 竹子和滴水石锥的碰撞问题（通过将它们设置为无服务端碰撞解决）

支持的版本：
- Spigot/Paper：1.14 - 1.21.8
- Fabric：1.21 - 1.21.4
