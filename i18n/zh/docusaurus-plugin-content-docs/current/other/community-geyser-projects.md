以下是一份由社区整理的、可与 Geyser 配合使用的插件、扩展及工具清单。

:::note

请注意，这些均为非官方项目，Geyser 官方 Discord 服务器可能无法为你遇到的任何漏洞或问题提供帮助。
请前往相关项目的链接页面，查看其对应的支持渠道。

:::

如果你希望将自己的插件加入此列表，请先前往 Geyser 的 #development（开发）频道沟通！

## BedrockFormShop {#bedrockformshop}

一款使用基岩版表单的 Spigot 商店插件。

* [GitHub](https://github.com/kejonaMC/BedrockFormShop)

创作者：[KejonaMC](https://github.com/kejonaMC)

## Bedrock Player Manager for Floodgate {#bedrock-player-manager-for-floodgate}
一款 Spigot 插件，可在基岩版玩家加入服务器时执行特定指令，也可让基岩版玩家绕过反作弊系统等。

* [Spigot 页面](https://www.spigotmc.org/resources/82278/)

创作者：ofunny

## BedrockSkinUtility {#bedrockskinutility}
这是一款 Fabric 模组，允许你在 Java 版中查看基岩版披风。

* [GitHub](https://github.com/Camotoy/BedrockSkinUtility)

创作者：[Camotoy](https://github.com/Camotoy)

## CreeperNation Skin API {#creepernation-skin-api}

一款皮肤接口（SkinAPI），可根据 UUID 或用户名显示 Java 版或基岩版皮肤。

* [官网](https://docs.creepernation.net)

创作者：[creeperz653](https://github.com/creeperz653)、[BlueTree242](https://github.com/BlueTree242)

## CrossplatForms {#crossplatforms}
一款适用于 Spigot、BungeeCord 和 Velocity 服务端的插件，可通过配置文件创建自定义的基岩版表单和物品栏菜单。
* [Spigot 页面](https://www.spigotmc.org/resources/101043/)
* [GitHub](https://github.com/kejonaMC/CrossplatForms)

创作者：[Konicai](https://github.com/Konicai)、[Jens](https://github.com/Jens-Co)

## CustomCommandPrefix {#customcommandprefix}
一款 Spigot 插件，允许用户输入以 `.` 开头的指令，以此规避 Xbox 成就被禁用的问题。

* [Spigot 页面](https://www.spigotmc.org/resources/87224/)
* [GitHub](https://github.com/ChewMC/CustomCommandPrefix)

创作者：[Chew](https://github.com/Chew)

## Emotecraft {#emotecraft}
一款模组（支持 NeoForge、Fabric）、插件（支持 Paper 服务端）及 Geyser 扩展，可让 Java 版玩家查看基岩版表情，同时让 Java 版和基岩版玩家都能使用自定义表情。

:::note

Java 客户端显然需要安装对应的模组才能生效。
你需要同时安装该扩展，以及适配你服务器的模组或插件。

:::

* [维基文档](https://docs.zigythebird.com/emotecraft/geyserext)
* [Modrinth（所有下载项）](https://modrinth.com/plugin/emotecraft)
* [CurseForge（Fabric 版）](https://www.curseforge.com/minecraft/mc-mods/emotecraft)
* [CurseForge（NeoForge 版）](https://www.curseforge.com/minecraft/mc-mods/emotecraft-forge)
* [Hangar](https://hangar.papermc.io/dima_dencep/emotecraft)
* [GitHub](https://github.com/KosmX/emotes)

创作者：[dima_dencep](https://github.com/dima-dencep)、[Zigy](https://github.com/ZigyTheBird)、[KosmX](https://github.com/KosmX)

## FloodgatePlaceholders {#floodgateplaceholders}
为 Floodgate 提供占位符支持的插件。

* [GitHub](https://github.com/rtm516/FloodgatePlaceholders)
* [eCloud](https://api.extendedclip.com/expansions/floodgate/)

创作者：[rtm516](https://github.com/rtm516)

## Floodgate-Skript {#floodgate-skript}
为 Skript 脚本语言提供 Floodgate 支持的插件。

* [GitHub](https://github.com/kejonaMC/floodgate-skript)

创作者：[Camotoy](https://github.com/Camotoy)

## GeyserBlockJavaPlayers {#geyserblockjavaplayers}
一款用于阻止 Java 版玩家加入你的 Geyser 服务器的插件，让你可以搭建仅允许基岩版玩家进入、且能使用 Spigot 插件的服务器。

* [GitHub](https://github.com/kejonaMC/GeyserBlockJavaPlayers)

创作者：[Camotoy](https://github.com/Camotoy)

## GeyserExtras {#geyserextras}
一款 Geyser 扩展/插件，旨在为基岩版玩家统一功能体验，包括优化战斗冷却的显示样式、战斗音效，以及其他实用功能和兼容方案。

* [Modrinth](https://modrinth.com/plugin/geyserextras)
* [Hangar](https://hangar.papermc.io/GeyserExtras/GeyserExtras)
* [GitHub](https://github.com/GeyserExtras/GeyserExtras)

创作者：[LetsGoAway](https://github.com/letsgoawaydev)

## GeyserPreventServerSwitch {#geyserpreventserverswitch}
阻止基岩版玩家加入代理服务器（Proxy）上的特定子服务器。

* [GitHub](https://github.com/kejonaMC/GeyserPreventServerSwitch)

创作者：[Camotoy](https://github.com/Camotoy)

## GeyserSkinManager {#geyserskinmanager}
一款适用于 Spigot 和 BungeeCord 服务端的插件，用于让基岩版皮肤正常显示；服务器上必须安装 Geyser 才能使用。
* [GitHub](https://github.com/Camotoy/GeyserSkinManager)

创作者：[Camotoy](https://github.com/Camotoy)

## GeyserUpdater {#geyserupdater}
这是一款适用于 Spigot/BungeeCord/Velocity 服务端的插件，可用于自动更新 Geyser。

* [GitHub](https://github.com/kejonaMC/GeyserUpdater)
* [Spigot 页面](https://www.spigotmc.org/resources/88555/)

创作者：[Jens](https://github.com/Jens-Co)、[YHDiamond](https://github.com/YHDiamond)、[Konicai](https://github.com/Konicai)

## SleepingServerStarter {#sleepingserverstarter}
一款轻量级应用，会等待玩家连接后再启动实际的 Minecraft 服务器。
* [GitHub](https://github.com/vincss/mcsleepingserverstarter)

创作者：[Vincss](https://github.com/vincss)

## Starlight Skin API {#starlight-skin-api}
一款简化基岩版皮肤输出流程的接口（API）。
* [官网](https://lunareclipse.studio/creations/starlight-skinapi)

创作者：[Bret06](https://www.bret06.net/)
