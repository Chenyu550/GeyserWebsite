全局 API 对所有服务器可用。它目前包括：[全局绑定](/wiki/floodgate/linking/#what-is-global-linking)、[皮肤上传](/wiki/floodgate/features/#what-is-skin-uploading)、通过玩家代号获取 xuid，以及通过 xuid 获取玩家代号。
除了您可以通过全局 API 访问的内容外，我们不会存储任何其他内容，除了一些不可识别的指标。全局 API 的源代码可在[这里](https://github.com/GeyserMC/global_api)找到，全局关联服务器的源代码可在[这里](https://github.com/GeyserMC/GlobalLinkServer)找到。

作为 API 的用户，您可以获取之前加入过任何 Geyser + Floodgate 服务器的 Geyser（或具体来说是 Floodgate 玩家）的各种信息。这包括：
- 从基岩版玩家代号转换为基岩版 xuid 的方法，反之亦然。仅为 Geyser 缓存中的玩家提供信息。<br />
  如果您需要所有玩家的信息，可以使用第三方 API，例如 https://mcprofile.io/endpoints，该 API 也从 Geyser 全局 API 提供信息。

- 皮肤信息：每当基岩版玩家通过 Floodgate 加入时，他们的皮肤会被转换并上传到 mineskin。如果您想获取该转换后的皮肤，在此端点中使用基岩版玩家的 xuid 来获取纹理 ID，或 base64 编码的"值"，您可以直接在玩家头颅等地方使用。
