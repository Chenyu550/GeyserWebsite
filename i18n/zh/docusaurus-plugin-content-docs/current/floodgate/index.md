# 概述
Floodgate 是一款混合模式的服务器插件。它带来了以下新功能：
- 客户端可使用我的世界基岩版账号加入我的世界Java版服务器，无需付费购买我的世界Java版账号。
- 支持在Java版中[显示基岩版玩家的皮肤](/wiki/floodgate/features#what-is-skin-uploading)。
- 允许插件向[基岩版客户端发送表单](/wiki/geyser/forms/)。
- 可通过[全局API](/wiki/api/api.geysermc.org/global-api/)查询玩家信息，
- 允许基岩版玩家通过[全局绑定](/wiki/floodgate/linking#what-is-global-linking)或本地绑定功能，将其账号与Java版账号进行绑定。

该插件需搭配 Geyser 一同安装使用。与 Geyser 不同的是，Floodgate 仅能以插件形式安装在 Spigot（包括 Paper 及其衍生服务端）、Bungeecord、Velocity 服务端上，或以模组形式安装在 Fabric 服务端上，且仅在安装了该插件/模组的服务器中生效。
