<Questions>
    <Q>
        如何更改或禁用基岩版玩家的前缀？
        <A>
            ***请注意：我们不建议移除前缀，除非你确定 Java 版和基岩版玩家之间不会有重名的情况。重名将导致一些奇怪的问题，例如无法传送到其中一名玩家。***

            在你的 Floodgate 配置文件中，将 `username-prefix` 更改为你想要的前缀——你可以将其设置为 `""` 以移除前缀。

            在一些较旧的 Paper 服务器（或基于其的衍生服务端）上，你可能还需要关闭服务器并删除与服务端 jar 文件同一目录下的 `usercache.json` 文件，以防止已加入过的玩家仍显示旧前缀。更多信息请查看 [此问题](/wiki/floodgate/issues#prefix-is-not-changing-on-the-server-after-changing-it-in-the-config)。
        </A>
    </Q>
    <Q>
        如何获取基岩版玩家的 UUID？
        <A>
            查看你的服务器日志，或使用 [此页面](https://mcprofile.io/)。如果这不起作用，可以尝试以下方法：

            首先，你需要获取玩家的 XUID。有一些第三方网站可以查询，例如 [这个网站](https://www.cxkes.me/xbox/xuid)（与 Geyser 无关）。请确保选择“Hexidecimal（十六进制）”选项。你需要输入玩家的 Xbox 游戏标签（Gamertag），提交后它会以 `xxxxxxxxxxxxxxxx` 的格式显示 XUID。要将 XUID 转换为 Java 版可以识别的 UUID，你需要将 XUID 格式化为：`00000000-0000-0000-xxxx-xxxxxxxxxxxx`。如果格式正确，Java 版将会接受它作为 UUID。
        </A>
    </Q>
    <Q>
        如何为基岩版玩家获取 PlaceholderAPI 的占位符？
        <A>
            如果你使用的是 Spigot 版本的 Floodgate，请从 [这里](https://github.com/rtm516/FloodgatePlaceholders/) 下载占位符插件。除了安装 [PlaceholderAPI](https://www.spigotmc.org/resources/6245/) 之外，使用这些占位符不需要额外的设置。如果你想在 BungeeCord 上使用，请查看上方关于在后端服务器上安装 Floodgate 的部分。
        </A>
    </Q>
    <Q>
        如何将 Skript 与 Floodgate 配合使用？
        <A>
            如果你使用的是 Spigot 版本的 Floodgate，有一个非官方插件可以添加 Skript 支持，下载地址：[这里](https://github.com/kejonaMC/floodgate-skript)。
        </A>
    </Q>
</Questions>
