description: Floodgate 的命令及其关联的权限节点。

| 命令 | 权限 | 描述 |
|:---:|:---:|:---:|
| `fwhitelist` | `floodgate.command.fwhitelist` | Floodgate 玩家的白名单命令。有关详细信息，请参阅[此处](/wiki/floodgate/features#whitelist-command)。 |
| `linkaccount` | `floodgate.command.linkaccount` | 在本地绑定账号。 |
| `unlinkaccount` | `floodgate.command.unlinkaccount` | 在本地解绑您的账号。 |
